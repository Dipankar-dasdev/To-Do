# 📝 Todo App

A simple Todo List application built using HTML, CSS and JavaScript.

## Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- ⌨️ Add tasks by pressing Enter
- 🎨 Clean and responsive interface

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

## What I Learned

- DOM manipulation
- Event listeners
- Creating dynamic elements
- Refactoring JavaScript into reusable functions
- Using `classList` to update the UI
